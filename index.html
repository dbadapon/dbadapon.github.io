Welcome to UTAOU!

About UTAOU!

UTAOU! (Japanese, "let's sing!") is a project that allows you to compose a song note-by-note (and syllable-by-syllable) for a virtual singer to sing. It is inspired by VOCALOID and UTAU, which are Japanese software programs that have similar functionality.

UTAOU! uses Japanese syllables becuase they are much simpler than those in other languages such as English.

Project and voice samples created by Dominique Adapon.

How to use UTAOU!

Song Display

Below this section is where your song will be displayed (after the label that reads "Song:"). A song is a sequential series of notes. Of course, the song is initally empty.

Each note in the song is represented as so:

[ syllable | note | value (duration) ]

To play the song, click the UTAOU! button.

Creating, Adding, and Deleting a Note

Below the song display is the note creator. There are four drop-down selectors where you can specify the Syllable, Note, Octave, and Duration of the note that you wish to add.

Below these drop-down selectors is a slider for controlling the intensity of the Vibrato for this note. If the slider is at 0, there will be no vibrato.

To preview the note that you have configured, click the [Preview] button to hear it.

To add this note to the song, click the [Add] button.

To delete the most recent note that you have added, click the [Delete Previous] button.

To clear the entire song, click the [Clear] button.

***NOTE: ADD, DELETE, AND CLEAR CANNOT BE UNDONE.
